# LogicTool

Инструмент для работы с булевыми функциями: таблицы истинности, DNF/KNF, проверка эквивалентности.

## Описание

LogicTool - это приложение на C#/.NET 8 с графическим интерфейсом (WPF/WinUI), предназначенное для анализа булевых функций. Проект реализован по методологии MVVM и состоит из 4 основных модулей.

## Функциональность

### Основные возможности:
- **Задание булевой функции** по номеру или по формуле
- **Построение таблицы истинности**
- **Генерация ДНФ и КНФ** в базисе {¬, ∧, ∨}
- **Проверка эквивалентности** двух функций
- **Подсчет "стоимости" формулы** (литералы, конъюнкты, дизъюнкты)

### Интерфейс:
- **Вкладка "По номеру"** - ввод числа переменных и номера функции
- **Вкладка "По формуле"** - парсинг и вычисление булевых выражений
- **Вкладка "Сравнение"** - проверка эквивалентности двух функций

## Структура решения

### LogicTool.Business
- `Interfaces` - интерфейсы сервисов
- `Services` - бизнес-логика приложения

### LogicTool.Core
- `Enums` - перечисления
- `Exceptions` - пользовательские исключения
- `Models` - модели данных
- `Services` - основные сервисы приложения

### LogicTool.Tests
- Модульные тесты (xUnit) для проверки:
  - Корректности таблиц истинности
  - Совпадения функций
  - Корректности ДНФ/КНФ

### LogicTool.UI
- `App.xaml` - точка входа приложения
- `MainWindow.xaml` - главное окно интерфейса
- Конвертеры и команды для WPF

- **Тестирование**: xUnit
